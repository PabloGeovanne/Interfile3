A aplicação concluída de forma satisfatória para a POC está contida nas pastas "Pab_Interfile_API" e "PUBLICAR/Pab_Interfile_Html/v2".

API

A API foi desenvolvida utilizando Entity Framework para a camada de banco de dados. Para depuração, pode ser necessário alterar a string de conexão e executar os comandos "Add-Migration" e "Update-Database".

Ambiente de Validação

  • Ambiente: IIS no Windows 11 Pro com SDK para .NET Core nas últimas versões.
  • URIs de Endpoints: As URIs dos endpoints precisarão ser ajustadas conforme abaixo:
    • API: http://api.interfile.local/index.html
    • HTML: http://html.interfile.local/index.html

Foi criada uma hospedagem local para simulação das URIs de requisição.
