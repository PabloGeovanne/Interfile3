Processo de Desenvolvimento

  • Habilitar o IIS:
    • Deixar desabilitado o WebDAV.
  • Instalar Asp.NET Core:
    • Instalar o Runtime, SDK e .NET Hosting para compatibilidade com .NET Core.
    • Verificar a instalação nos Módulos do IIS.
  • Criação do Projeto:
    • Criar um projeto Web API com C# no Visual Studio.
    • Criar um perfil de publicação no Visual Studio.
    • Publicar a solução API.
  • Configurações de Acesso:
    • Autorizar o acesso de todos os usuários do Windows à pasta dos arquivos.
  • Configurações do IIS:
    • Adicionar um pool de aplicação (cache dos dados da aplicação).
    • Adicionar um site no IIS e realizar as configurações necessárias.
  • Configurações do Projeto:
    • Adicionar configurações do Swagger no arquivo Program.cs.
    • Adicionar páginas HTML de termos de licença e página para contato.
    • Configurar uma subpasta no IIS para compilar as páginas HTML no pool padrão.
  • Configurações de Banco de Dados:
    • Adicionar a string de conexão no arquivo appsettings.json.
    • Criar a pasta MODELS para separar as classes de objetos.
    • Criar a pasta DATA para separar as classes de banco de dados.
    • Instalar o NuGet da Microsoft.EntityFrameworkCore.
    • Criar a classe Models/Produto para representar o objeto usado no CRUD.
    • Criar a classe ApplicationDbContext para definir o contexto de banco de dados ao Entity Framework.
    • Configurar o arquivo Program.cs para utilizar recursos do Entity Framework com o SQL Server.
    • Criar um controller de Produto e os endpoints RESTful.
    • Criar o banco de dados no SQL Server.
    • Rodar os comandos Add-Migration InitialCreate e Update-Database.
    • Verificar a tabela criada no banco de dados.
    • Testar os endpoints com o Swagger inserindo, lendo, atualizando e excluindo dados.
    • Testar operações na tabela correspondente.
    • Ajustar a instância do banco de dados para criação de usuário e permissionamento do Pool do IIS para autenticação do Windows.
  • Desenvolvimento do Front-end:
    • Criar um projeto Angular com o terminal PowerShell.
    • Criar um projeto HTML com Bootstrap e JavaScript/jQuery, realizando requisições na API .NET Core e seus respectivos endpoints para CRUD no banco de dados.
  • Implementação de Autenticação:
    • Implementar autenticação com JWT.
    • Gerar chave secreta randomicamente com 32 bytes em hexadecimal utilizando Node.js para JWT.
    • Criar a classe Models/Pessoas.
    • Rodar os comandos Add-Migration e Update-Database.
    • Implementar o algoritmo JavaScript/jQuery para realizar o CRUD de Pessoas.
    • Implementar a lógica de JWT Token para autenticação e autorização no consumo dos endpoints com JavaScript (90% concluído).
    • Validar os endpoints de Pessoa e Produtos.
